# Plenary（設備手帖）

**空調・衛生設備エンジニアのための、ブラウザだけで使える計算ツール集**

🔗 **公開ページ: https://meyonze.github.io/Plenary/**

インストール不要・登録不要。URLを開くだけで、現場でもデスクでもすぐに計算できます。PWA対応のため、スマートフォンのホーム画面に追加すればアプリのように使えます。

---

## 収録ツール（全14種）

空調・衛生設備の実務でよく使う計算を、単一HTMLファイルにまとめています。

### 空調計算

- **ダクトサイズ計算** — 低速・主ダクト／高速ダクト／高圧1・2ダクト等、風速区分ごとの最適ダクト寸法を算出。塩ビダクトなど材質ごとの粗度係数にも対応。
- **換気量計算** — 必要換気量・換気回数法による計算。1人当たり換気量や実換気回数の算出にも対応。
- **空気線図（湿り空気線図）** — 乾球温度・湿球温度・相対湿度・露点温度の組み合わせから、比エンタルピーなど湿り空気の状態値を算出。
- **制気口計算** — 制気口有効面積の算定。
- **負荷計算（簡易）** — 簡易熱負荷の概算。
- **風速・動圧・圧力損失変換** — ダクト内風速・動圧・圧力損失の相互変換。
- **単位換算** — 空調・設備計算で頻出する各種単位の換算。

### 音響・騒音計算

- **屋外騒音計算（前川チャート）** — 前川の点音源式に基づく回折減衰計算。建物による音の回り込みを考慮した屋外騒音レベルの予測に対応。
- **騒音のエネルギー合成** — 複数の騒音源を合成した際の合成騒音レベルを計算。
- **NC判定** — 室内騒音レベルからNC値を判定。

### 給排水・配管計算

- **水配管計算** — 配管径・配管長さ・横走り配管長さから圧力損失等を算出。
- **排水勾配計算** — 用途別の最小勾配目安をもとに、採用勾配・配管実長から排水計画を支援。

### 支持金具・鋼材計算

- **吊りボルト計算** — 吊りボルト本数・間隔・振れ止め箇所数の算定を支援。
- **鋼材重量計算** — JIS G3192（熱間圧延形鋼）・JIS G3350（軽量形鋼）データに基づく、山形鋼・溝形鋼・軽溝形鋼・リップ溝形鋼などの断面・重量計算。複数鋼材の組み合わせ小計にも対応。

## 特徴

- **インストール不要**：URLを開くだけで即利用可能。ブックマークしておけばいつでもアクセスできます。
- **オフライン対応（PWA）**：ホーム画面に追加すると、電波の弱い現場でも動作します。
- **マルチプラットフォーム**：PC・iPhone・Androidいずれの端末でも同じ画面で利用できます。
- **入力データはブラウザ内のみ**：計算内容が外部サーバーに送信されることはありません。

## 対象ユーザー

設備設計者・施工管理者・現場のHVAC技術者・建築設備の積算担当者など、空調・衛生設備にまつわる計算を日常的に行う方を想定しています。
